# 笔记工作流

> 快速记录 → 整理归档 → 建立连接 → 提炼复用

## 四步工作流

### 1. 捕获 (Capture)

- 随时记录想法到 [[Daily Notes|每日笔记]]
- 使用 #quick-note 标签标识快速笔记
- 不要在意格式和完整性

### 2. 整理 (Organize)

- 每天/每周回顾快速笔记
- 将精华内容拆解为原子笔记
- 移动到对应分类文件夹
- 去掉 #quick-note 标签

### 3. 连接 (Connect)

- 为新笔记添加 `[[Wikilink]]` 到相关笔记
- 更新 MOC 页面
- 在 Graph View 中检查连接性
- 添加标签和分类

### 4. 提炼 (Distill)

- 定期 Review 笔记
- 将使用频率高的知识提炼为技能 (Skill)
- 将重要发现整理为研究报告
- 归档不再需要的笔记
