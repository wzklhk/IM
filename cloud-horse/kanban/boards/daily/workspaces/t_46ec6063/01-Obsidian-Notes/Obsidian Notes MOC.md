# Obsidian 笔记总览

> 📅 创建时间: 2026-05-10

## 关于本知识库

本知识库服务于 Hermes Agent 的个人知识管理 (PKM) 系统，涵盖三个核心领域：

1. **Obsidian 笔记** — 日常笔记、思考记录、知识卡片
2. **技能文档** — Hermes Agent 技能文档 (Skills) 的管理与维护
3. **研究报告** — 技术研究报告、论文摘要、调研笔记

## 笔记分类

| 分类 | 说明 | 标签 |
|------|------|------|
| 快速笔记 | 闪念想法、临时记录 | #quick-note |
| 知识卡片 | 原子化知识单元 | #card |
| 项目笔记 | 项目相关记录 | #project |
| 学习笔记 | 学习新技术的笔记 | #learning |
| 会议纪要 | 会议记录与行动项 | #meeting |
| 日常日志 | 每日记录 | #daily |

## 笔记原则

- **原子性**: 每条笔记只包含一个核心概念
- **链接性**: 使用 `[[Wikilink]]` 连接相关笔记
- **可发现**: 通过 MOC (Map of Content) 组织导航
- **渐进式**: 从快速笔记 → 永久笔记 → 知识卡片
