# 日语假名与罗马字对照表

> 日语（日本語）使用三种文字系统：平假名（ひらがな）、片假名（カタカナ）、汉字（漢字）。
> 拉丁转写采用 **平文式罗马字**（Hepburn Romaji），日本国内也使用训令式。

## 五、特殊音

| 名称 | 标记 | 说明 | 例子 |
|:----:|:----:|:----|:----:|
| **促音** | っ/ッ (小字) | 双写后续辅音 | がっこう (gakkō)、マッチ (matchi) |
| **长音** | ー (片假名) / あ段+あ、等 | 元音拉长一倍 | 東京 (Tōkyō)、ケーキ (kēki) |
| **拨音** | ん/ン | 鼻音 | にほん (nihon)、テレビ (terebi) |

### 长音标记规则（平假名）

| 段 | 长音拼写 | 例子 |
|:-:|:--------:|:----:|
| あ段 | あ段假名 + あ | おかあさん (okāsan) |
| い段 | い段假名 + い | おにいさん (onīsan) |
| う段 | う段假名 + う | くうき (kūki) |
| え段 | え段假名 + い/え | せんせい (sensei)、おねえさん (onēsan) |
| お段 | お段假名 + う/お | とうきょう (Tōkyō)、おおきい (ōkii) |

### 促音罗马字拼写

| 促音形式 | 罗马字 |
|:--------:|:------:|
| っ+か行 | **kk** (がっこう → gakkō) |
| っ+さ行 | **ss** (いっさい → issai) |
| っ+た行 | **tt** (マッチ → matchi) |
| っ+ぱ行 | **pp** (いっぱい → ippai) |
| っ+ち | **tch** (いっち → itchi) |
| っ+つ | **tts** (みっつ → mittsu) |
