# 🏖️ 咖啡师FIRE路线图

> 不提前退休，但要提前拥有"做自己喜欢的事"的自由

---

## 核心定义

**咖啡师FIRE ≠ 不工作**
**咖啡师FIRE = 被动收入覆盖部分生活 → 你可以选择低压力但喜欢的工作**

```
你现在的剧本：
  被动收入（3,300/月）→ 覆盖广州房租的~
  AI研发工作（主业收入）→ 覆盖剩余开销 + 继续存钱
  100万安全垫 → 不焦虑被优化、不焦虑薪资低
```

## 三阶段路线

### 第一阶段：广积粮（2026-2027）

| 时间 | 目标 | 动作 |
|------|------|------|
| 2026.06 | 达卡结束 | 结清当地事宜 |

**100万目标分解：**
```
当前：80万

投资收益（5-8%）：+3-5万
────────────────
预计2027.06：~103-115万 ✅
```

### 第二阶段：转型积累（2027-2030）

| 年份 | 预期资产 | 场景假设 |
|------|---------|---------|
| 2027 | 100-120万 | AI岗入职广州，月薪~2万 |

| 2029 | 160-180万 | 稳定积累 |
| 2030 | 190-220万 | 接近200万大关 |

**广州AI岗的收入假设（保守）：**
```

月支出（两人）：~8K-10K
月存：~8K-10K
年存：~10-12万
加上投资复利 → 3年从100万到~150万
```

### 第三阶段：咖啡师FIRE舒适区（2031+）

| 资产 | 月被动收入（4%） | 覆盖什么 |

> **咖啡师FIRE触发线：200万**

## 注意事项

- **不要因为到了100万就放松储蓄** — 100万只是安全垫，不是终点
- **结婚后建议两人账户分开** — 各管各的投资，共同开销AA或比例分摊
- **买房会大幅影响进度** — 在广州买房可能要花掉50-80万首付，FIRE进度倒退3-5年
- **孩子是最大的变量** — 如果有小朋友计划，FIRE目标至少加200万
- **保持医保不断缴** — 哪怕换工作也要自己续上
