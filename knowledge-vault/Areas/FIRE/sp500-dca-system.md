## 一、核心规则

```
每月基准：500股（雷打不动）
动态调整：看标普500指数月度涨跌
```

### 每月22号操作表

| 当月涨跌幅（对比上月22号） | 买入量 | 说明 |
|--------------------------|--------|------|
| 当月跌 > 10% | **1,000股** 🟢 | 大跌大买，别人恐惧我贪婪 |
| 当月跌 5%-10% | **750股** 🟢 | 小跌多买，打折加仓 |
| 当月涨跌在 ±5% 内 | **500股** ⚪ | 正常定投，不动如山 |
| 当月涨 > 5% | **300股** 🔴 | 涨了少买，不追高 |
| 连跌3个月以上 | 每月再加200股（=700/950/1,200） | 恐慌区加大力度 |
| 连涨3个月以上 | 每月减200股（=100/300/300） | 狂热区控制仓位 |

### 一句话记忆

> **跌多买多，涨多买少，不涨不跌正常买。**

## 四、为什么这个量化系统有效

```
普通定投：每月500股，雷打不动
  → 平均成本 = 期间均价

量化定投：根据涨跌调整股数
  → 跌的时候买得多 → 拉低平均成本
  → 涨的时候买得少 → 不追高
  → 长期下来，成本低于期间均价
  
这就是"低买高卖"在定投中的量化实现。
```
